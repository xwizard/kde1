4. Questions and answers

4.1 I get the message "Your system is currently not supported.". What can I do?

KTop will compile on many UNIX systems but it can only be used on some. If your system is not yet supported you will get this message when running KTop. You can download the sources from the KDE CVS or the KTop Homepage. Look at the OS*.h classes and implement support for your OS. Please contact me in advance to avoid duplicated efforts. Your help is greatly appreciated.

4.2 I have a multi processor box but I don't get the additional meters for each CPU. Why not?

Your operating system kernel needs to provide this information. If it is not available KTop cannot display the load for each processor. Currently only Linux 2.2 kernels provide this information. Linux 2.0 or earlier kernels do not provide the required information.
