Introduction

Hex Editor is a small and simple viewer for binary files. I borrowed the DnD support of KEdit for this editor, so he now can handle URLs with kfm.

Changes

Version 0.4 (Stephan Kulow)

  • A little help and about
  • Save and SaveAs
  • Editing supported (0.4 is enough :-)
  • bugfixes

Version 0.3 (Stephan Kulow)

  • changed the type of window handling (more files => more windows)
  • scrolling handled smoother
  • added Drag and Drop
  • added network access.
  • usage of libkdecore and libkfm
  • bugfixes
