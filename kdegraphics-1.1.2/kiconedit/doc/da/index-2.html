2. Installation

2.1 Hvor finder jeg Ikon Editoren

KDE Ikon Editoren kan findes på ftp://ftp.kde.org/pub/kde/unstable/apps/graphics, KDE projektets primære site eller på min hjemmeside.

2.2 Systemkrav

For succesfuldt at kunne kompilere ikon editoren, har du brug for enten kdelibs-1.0 eller de nyeste versioner af libkdecore, libkdeui, libkfile og libkfm. Alle krævede libraries så vel som KDE Ikon Editoren selv kan findes på ftp://ftp.kde.org/pub/kde/.

2.3 Kompilering og installering

For at kompilere og installere KDE Ikon Editoren på dit system, tast følgende i det directory hvor du har udpakket Ikon Editor distributionen: 

 
% ./configure 
% make 
% make install

Da Ikon Editoren bruger autoconf og automake skulle du ikke få problemer med at kompilere den. Skulle det alligevl volde problemer så rapporter det venligst til KDE mailing listen.
