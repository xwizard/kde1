3. Configuration

All options are saved in ksysvrc. Defaults are taken from the global file, if available, or else generated within KSysV. Changes to the defaults are stored in a local ksysvrc, normally $HOME/.kde/share/config/ksysvrc.

3.1 Recognized Sections

Recognized sections and keys plus their default values.

[Path Settings]

  • ScriptPath (default: /etc/rc.d/init.d): path to the scripts used for starting and stopping services
  • RunlevelPath (default: /etc/rc.d): path to the runlevel sub-directories

[Colors]

  • Changed (default: red): color used for changed entries
  • New (default: blue): color used for new entries

[Geometry]

  • Width (default: none): width of KSysV´s window
  • Height (default: none): height of KSysV´s window

[Other Settings]

  • ToolBar (default: true): whether the toolbar is enabled
  • StatusBar (default: true): whether the status bar is enabled
  • ShowLog (default: true): whether the logfile is shown
  • PanningFactor (default: 80): 100 - PanningFactor = percentage of window reserved for the logfile display
