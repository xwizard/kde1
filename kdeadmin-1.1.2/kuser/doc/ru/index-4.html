4. Настройка

4.1 Редактирование параметров создания нового пользователя

Для редактирования параметров создания нового пользователя вы можете использовать диалог `редактировать параметры по умолчанию', который доступен через меню как Файл->Настройки. Здесь вы можете изменить параметры, используемые при создании нового пользователя: оболочку, базовый каталог для личных каталогов, создавать или нет личный каталог, копировать ли скелет личного каталога и использовать ли схему личных групп.
