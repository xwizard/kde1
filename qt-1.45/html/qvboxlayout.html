Detailed Description

The QVBoxLayout class lines up child widgets vertically.

This class provides an easier way to construct vertical box layout objects. See QBoxLayout for more details.

The simplest way to use this class is: 

     QBoxLayout * l = new QVBoxLayout( widget );
     l->addWidget( aWidget );
     l->addWidget( anotherWidget );
     l->activate()

See also: QHBoxLayout and QGridLayout.

Examples: layout/layout.cpp widgets/widgets.cpp
