Detailed Description

The QHBoxLayout class lines up child widgets horizontally.

This class provides an easier way to construct horizontal box layout objects. See QBoxLayout for more details.

The simplest way to use this class is: 

     QBoxLayout * l = new QHBoxLayout( widget );
     l->addWidget( aWidget );
     l->addWidget( anotherWidget );
     l->activate()

See also: QVBoxLayout and QGridLayout.

Examples: layout/layout.cpp widgets/widgets.cpp
