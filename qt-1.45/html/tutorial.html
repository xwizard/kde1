Tutorial: The 14 Steps


This tutorial gives an introduction to GUI programming using the Qt toolkit 1.44 . It doesn't cover everything: The emphasis is on teaching the programming philosophy of GUI programming, and Qt's features are introduced as needed.

Chapter one starts with a ten-line hello-world and each subsequent chapter introduces one or a few more concepts. By chapter 14, the ten lines from chapter one have turned into a 650-line game.

Tutorial chapters:

  1. Hello, World!
  2. Calling it Quits
  3. Family Values
  4. Let There Be Widgets
  5. Building Blocks
  6. Building Blocks Galore!
  7. One Thing Leads to Another
  8. Preparing for Battle
  9. With Cannon You Can
  10. Smooth as Silk
  11. Giving It a Shot
  12. Hanging in the Air the Way Bricks Don't
  13. Game Over
  14. Facing the Wall
