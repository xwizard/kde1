1. Introduction

KSame is a simple game. It's played by one player, so there is only one winner :-) You play for fun and against the highscore. It has been inspired by SameGame, that is only famous on the Macintosh plattform.

Last year Stephan Kulow and I wrote "Probiere" (a German pun) and it was our first attempt to write a X11 application. We decided that it would be a good idea to port it to KDE and I've done it with some help of Stephan.
