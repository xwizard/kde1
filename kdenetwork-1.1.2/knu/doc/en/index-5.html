5. The Finger tab

5.1 What is it?

The Finger tab allows the execution of the finger(1) command. The finger command displays information about the system users.

The command output will depends on the server which will make the reply.

5.2 Options

Server

This is the name (or the address) of the finger server to ask.

5.3 History The finger command appeared in 3.0BSD.
