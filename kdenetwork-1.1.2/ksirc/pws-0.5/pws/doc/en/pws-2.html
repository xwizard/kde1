2.3 Compilation and installation

In order to compile and install PWS on your system, type the following in the base directory of the PWS distribution: 

% ./configure
% make
% make install

Since PWS uses autoconf you should have not trouble compiling it. Should you run into problems please report them to the the author at ralsina@unl.edu.ar
