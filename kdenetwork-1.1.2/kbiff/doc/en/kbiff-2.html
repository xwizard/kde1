2. Installation

2.1 How to obtain KBiff

The most current release version in both source and binary formats can always be found at the KBiff homepage. This is at http://www.pobox.com/~kurt_granroth/kbiff KBiff is also now part of the KDE Networks package. As such, the bleeding edge release can always be found in the KDE CVS repository.

2.2 Requirements

In order to successfully compile KBiff, you need at least the 1.0 version of KDE. All required libraries can be found on ftp://ftp.kde.org/pub/kde.

2.3 Compilation and Installation

Compiling KBiff is very easy. The following should do it: 

% ./configure
% make
% make install

That should do it! Should you run into any problems, please report them to the author
