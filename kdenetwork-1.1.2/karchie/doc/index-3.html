3. Getting KArchie started

This section gives you a small introduction to the first usage of KArchie. More information can be obtained in the Features section.

3.1 First view

After starting the programm, you will see the main window with a single line editor, the empty file list, where the results will be stored, and the file view with a more detailed discription of the currently selected file.

3.2 Start query

To start an initial query, simply type in the name of the programm you are searching in the Search Term field and press ENTER. In the statusbar below you see the programm status changing to query running and the number of retries or hits increasing.

3.3 Query results

If your search has been successfull, you see a full list of files and a detailed file description in the description field below. You may now select a file in a FTP server near to your host and open the server directory with menu File->OpenDir or download the file to your computer with menu File->Download..
