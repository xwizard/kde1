2. Installation

2.1 How to obtain KArchie

KArchie is a part of the KDE project http://www.kde.org. It can be found on ftp://ftp.kde.org/pub/kde/, the main ftp site of the KDE project, in the kdenonbeta snapshots. Please consider using your nearest mirror ftp site to get the load away from the server.

2.2 Compilation and installation

In order to compile and install KArchie on your system, type the following in the base directory of the KArchie distribution: 

% ./configure
% make
% make install

Since KArchie uses autoconf you should have not trouble compiling it. Should you run into problems, please report them to my email address j.habenicht@europemail.com .
