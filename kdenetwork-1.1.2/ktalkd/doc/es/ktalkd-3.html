3. Instalación

3.1 ¿Cómo obtener Ktalkd?

Ktalkd es una aplicación principal del proyecto KDE http://www.kde.org, parte del paquete kdenetwork.

Puedes bajarte Ktalkd del servidor principal de ftp del proyecto KDE, ftp://ftp.kde.org/pub/kde y de sus mirrors.

3.2 Requisitos.

Para compilar correctamente Ktalkd, necesitas las últimas versiones de las librerías de KDE así como las librerías Qt C++. Todas las librerías necesarias así como el propio Ktalkd se pueden encontrar en ftp://ftp.kde.org/pub/kde/.

3.3 Compilación e Instalación.

Para compilar e instalar Ktalkd en tu sistema, teclea lo siguiente en el directorio principal de la distribución de ktalkd: 

% ./configure
% make
% make install

Como Ktalkd es un demonio, make install necesitará privilegios de root.

No olvides actualizar /etc/inetd.conf. Por ejemplo en un sistema linux, si kde es en /opt/kde, cambia las lineas correspondientes a talk y ntalk a: 

talk   dgram   udp     wait    root    /usr/sbin/tcpd  /opt/kde/bin/kotalkd
ntalk  dgram   udp     wait    root    /usr/sbin/tcpd  /opt/kde/bin/ktalkd

Proporcionamos un script para hacer los cambios necesarios de forma automática. Actualiza tu archivo inetd.conf tecleando simplemente 

% ./post-install.sh
De todas formas, deberás reiniciar el inetd después de ésto. En la mayoría de los sistemas LiNUX, haz : 
% killall -HUP inetd

Por favor infórmame de cualquier modificación que hayas tenido que hacer para compilar Ktalkd en tu plataforma.
