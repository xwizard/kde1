3. Installation

3.1 Comment obtenir ktalkd

ktalkd est à présent une application centrale du projet KDE http://www.kde.org, faisant partier du package kdenetwork.

Vous pouvez décharger ktalkd depuis le site principal du projet KDE, ftp://ftp.kde.org/pub/kde et depuis ses sites miroirs. Il se trouve d'habitude dans ftp://ftp.kde.org/pub/kde/stable/latest/apps/network.

3.2 Configuration nécessaire

Il vous faut les dernières versions des bibliothèque KDE ainsi que la librairie C++ Qt pour compiler ktalkd avec succès. Toutes les librairies nécessaires ainsi que ktalkd lui-même peuvent être trouvés sur ftp://ftp.kde.org/pub/kde/.

3.3 Compilation et installation

Pour compiler et installer ktalkd sur votre système, tapez les commandes suivantes dans le répertoire de base de la distribution de ktalkd : 

% ./configure
% make
% make install

Comme ktalkd est un démon, make install aura besoin des privilèges de l'utilisateur 'root'.

N'oublize pas de mettre à jour /etc/inetd.conf. Par exemple, sur un système linux, si kde se trouve dans /opt/kde, changez les lignes concernant talk et ntalk en : 

talk   dgram   udp     wait    root    /usr/sbin/tcpd  /opt/kde/bin/kotalkd
ntalk  dgram   udp     wait    root    /usr/sbin/tcpd  /opt/kde/bin/ktalkd
Un script permettant d'effectuer les modifications nécessaires automatiquement est fourni. Mettez à jour votre fichier inetd.conf en exécutant simplement : 
% ./post-install.sh
Quelle que soit la méthode choisie, vous devrez redémarrer inetd ensuite. Sur la plupart des systèmes Linux, faites : 
% killall -HUP inetd

Veuillez m'informer de toute modification que vous auriez eue à accomplir pour réussir à compiler ktalkd sur votre plate-forme.
