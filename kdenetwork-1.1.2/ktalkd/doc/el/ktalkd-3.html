3. Εγκατάσταση

3.1 Πως να αποκτήσετε το ktalkd

Το ktalkd είναι πλέον βασική εφαρμογή του έργου KDE http://www.kde.org, μέρος του πακέτου kdenetwork .

Μπορείτε να κατεβάσετε το ktalkd από το κύριο ftp site του έργου KDE, ftp://ftp.kde.org/pub/kde ή από τα mirrors του .

3.2 Απαιτήσεις

Για να μεταγλωττιστεί το ktalkd επιτυχώς, χρειάζεστε την τελευταία έκδοση των βιβλιοθηκών του KDE όπως επίσης και την βιβλιοθήκη Qt για C++ . 1/4λες οι απαιτούμενες βιβλιοθήκες καθώς επίσης και το ίδιο το ktalkd μπορούν να βρεθούν στο ftp://ftp.kde.org/pub/kde/.

3.3 Μεταγλώττιση και Εγκατάσταση

Για να μεταγλωττίσετε και να εγκαταστήσετε το ktalkd στο σύστημά σας, γράψτε τα παρακάτω στο βασικό κατάλογο της διανομής του ktalkd : 

% ./configure
% make
% make install

Καθώς το ktalkd είναι δαίμονας, το make install θα απαιτήσει δικαιώματα root.

Μην ξεχάσετε να ενημερώσετε το /etc/inetd.conf. Για παράδειγμα, σε ένα σύστημα linux, αν το kde είναι στο /opt/kde, αλλάξτε τις γραμμές που σχετίζονται με το talk και το ntalk σε : 

talk   dgram   udp     wait    root    /usr/sbin/tcpd  /opt/kde/bin/kotalkd
ntalk  dgram   udp     wait    root    /usr/sbin/tcpd  /opt/kde/bin/ktalkd
Σας παρέχεται επίσης ένα script, που κάνει αυτόματα τις απαραίτητες αλλαγές. Ενημερώστε το inetd.conf αρχείο σας τρέχοντας απλά 
% ./post-install.sh
1/4πως και να έχει, θα πρέπει να επανεκκινήσετε το inetd μετά από αυτό. Στα περισσότερα συστήματα Linux, γίνεται με : 
% killall -HUP inetd

Παρακαλώ ενημερώστε με για οποιαδήποτε τροποποίηση που χρειάστηκε να κάνετε για να μεταγλωττίσετε το ktalkd στην πλατφόρμα σας.
