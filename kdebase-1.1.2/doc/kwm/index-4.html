4. Sound Events

KWM raises a couple of sound events, which can be used in a system sound module. At present the following sound events are implemented:

  • Window Iconify
  • Window DeIconify
  • Window Maximize
  • Window UnMaximize
  • Window Sticky
  • Window UnSticky
  • Window New
  • Window Delete
  • Window New Trans
  • Window Delete Trans
  • Window Shade Up
  • Window Shade Down
  • Logout
  • Logout Message
