1. Introduction

kpanel helps you start and manage your applications and virtual desktops in KDE. It provides quick access to a number of things, including:

  • the KDE application starter
  • a taskbar of running applications
  • buttons that launch frequently used applications
  • virtual desktops

kpanel is highly flexible and configurable, and supports both system-wide and personalized user settings. This document describes how to configure and use kpanel.

1.1 Features

Some of the features provided by kpanel include:

  • independently positionable panel and taskbar
  • auto-hide capability for panel and taskbar
  • configurable tooltips
  • ability to run small notification/status apps right in the panel
  • ability to define multiple icons for a button on the panel, and switch between them using kpanel commands
