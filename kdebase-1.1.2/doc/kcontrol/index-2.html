2. The Configuration Modules

This is a list of the standard configuration modules provided by the KDE base package. Please note that there may be many more modules on your system if you install additional software. You can access the help for any module by first selecting the module in the control center and the pressing the help button.

  • Bell Configuration
  • Memory Information
  • Processor Information
  • IRQ Information
  • DMA Channel Information
  • I/O Port Information
  • PCI Information
  • Soundcard Information
  • Device Information
  • SCSI Information
  • Partition Information
  • X-server Information
  • The Samba Status Monitor
  • Keyboard Configuration
  • Mouse Configuration
  • Configuration of Key Bindings (Shortcut keys)
  • Titlebar Button Configuration
  • Titlebar Configuration
  • Window Configuration
  • File Manager Configuration: Fonts
  • File Manager Configuration: Color
  • File Manager Configuration: Other settings
  • Web Browser Configuration: Proxy Servers
  • Web Browser Configuration: HTTP Settings
  • Web Browser Configuration: User Agent String
  • Web Browser Configuration: Cookies
  • Configuration of Placement of Desktop Icons
  • Language Setting
  • System Sound Selection
  • Panel and Taskbar Configuration
  • Background Configuration
  • Screen Saver Configuration
  • Color Scheme Configuration
  • Style Configuration
