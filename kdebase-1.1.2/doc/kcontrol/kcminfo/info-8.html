8. Information about the available devices

This page displays information about the available devices,

The exact information displayed is system-dependent. On some systems, device information cannot be displayed yet.

On Linux this information is read from /proc/devices and /proc/misc, which are only available if the /proc pseudo-filesystem is compiled into the kernel. Devices are listed by group (Character, Block, or Miscellaneous), and by major numbers.

The user cannot modify settings on this page.
