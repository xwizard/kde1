1. Memory usage information

This page displays information about the current memory usage. It is updated regularly.

The exact information displayed is system-dependent. On some systems, memory usage information cannot be displayed yet.

On Linux this information is read from /proc/meminfo, which is only available if the /proc pseudo-filesystem is compiled into the kernel.

The user cannot modify settings on this page.
