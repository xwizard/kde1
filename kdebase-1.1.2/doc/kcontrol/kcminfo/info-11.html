11. Information about the X-Server that is being used.

This page displays information about the X-Server.

The exact information displayed is system-dependent. On some systems, X-Server information cannot be displayed yet.

On most KDE-Platforms this dialog shows the name of the X11-display, the X11 version number, a vendor string, the vendor release number, the default screen number and the number of available screens. Furthermore this dialog also shows the dimensions in pixels and centimeters, the resolution in dots per inch (dpi), and the available color-depths for every screen.

The user cannot modify settings on this page.
