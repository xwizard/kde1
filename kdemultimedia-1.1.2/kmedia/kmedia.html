kmedia - the KDE Multimedia player

This documentation has been updated to match Version 0.40.

Introduction

kmedia is a program that can play various types of multimedia objects, such as audio files (*.wav), midi music (*.mid), MPEG videos (*.mpg, *.mpeg) and Soundtracker modules (*.mod, mod.*).

This is an alpha release of kmedia. Wave audio, Midi tunes and Soundtracker modules are currently supported.

Usage:

  1. Just start kmedia from kpanel or by double-clicking a desktop icon.
  2. Drag one Wav from kfm into the kmedia window. The Wav should play now.
  3. Press "Stop" or "Pause" to control playback.
  4. You can drag another Wav to the same window, if you like.

The media players

    The media players are programs, that can play certain media types. There is one program available for playing wave audio. Soon there will be one for midi playing, Mod playing and so on. Please note the documentation below.

    The player programs below are all copyrighted by their authors. Please contact them only, if you have problems with the players. Before contacting them, please try the original program, as these versions are patched by me.

    Contact me, if you have problems with kmedia, maudio and the mediatool library. You should contact me too, when you have no trouble with the original player program, but with the patched one.

BUGS:

  • kmedia and maudio eat up CPU time even when being "idle". This problem and it solution is known already.
  • Multiple window support does not work 100%. Closing the application when multiple windows were open can leave "maudio" processes running, which you have to kill explicitely.
