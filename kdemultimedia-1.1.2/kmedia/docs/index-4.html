4. The Menu Entries

4.1 File

Open...

Opens a media. A file selector pops up and asks for a Media.

New view

Open new media player window

Exit

Closes this media player

Quit

Quits the Mixer.

4.2 Options

Mixer

Opens a mixer window. You can set the mixer you want in the preferences dialog

Preferences

Opens the KMedia options dialog

4.3 Help

Help

Invokes the KDE help system with the KMedia help pages displayed.

About kmedia...

Displays essential information about KMedia.
