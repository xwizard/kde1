3. Onscreen Fundamentals

KMedia usage is simple. Start the program and drag a WAV file into the window. The file will be played immediately. If the file does not get played your sound device might be busy.

3.1 Buttons

The meaning of all buttons should be well known. If you can operate a tape deck or CD Player, you should have no problem in using KMedia. Only applicable buttons are usable. For example, ejecting a CD makes sense, but not ejecting a WAV Media.

3.2 Multiple media players

You can open multiple KMedia windows. Either by using File->New view or by starting another media player through kpanel.

If you want KMedia to use a specific output device (e.g. soundcard) you can set it in kmedia's configuration dialog. Attention: This is currently unimplemented!!! You can set the device using numbers. 0 means the default device, 1 the primary, 2 the secondary device and so on.

3.3 Key Bindings

KMedia honors the following key binding.

  • Control-N Open new media player window
